# Game Settings Panel

A lightweight, visually appealing Game Settings UI component designed for seamless integration into web-based games or FRC dashboards.

## ‚ú® Features
* **Toggle Options:** Easily enable or disable Sound Effects and Background Music.
* **Game Modes:** Integrated "Hard Mode" switch for difficulty adjustments.
* **Haptic Support:** Toggle for vibration/haptic feedback on supported devices.
* **Clean UI:** Soft shadows and rounded corners for a modern aesthetic.

## Usage
To use this panel in your project:
1. Clone the repository.
2. Import the `settings-panel` component.
3. Link your game state to the checkbox listeners.
